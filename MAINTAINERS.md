# Maintainers

This file lists the maintainers of the Volt open source projects, as referenced by [`GOVERNANCE.md`](GOVERNANCE.md). It is the org-wide default; a repository may ship its own `MAINTAINERS.md` listing repo-specific maintainers, which takes precedence for that repo.

The two-owners-per-repo `CODEOWNERS` target kicks in as the team grows; today the org runs with a single interim maintainer (below).
