# Support

Thanks for using Volt. Here is where to get help, fastest path first.

## Documentation

- **Product & API docs:** https://docs.voltcloud.ai
- **API reference:** generated from the OpenAPI spec, linked from the docs site.
- **SDK guides:** in each SDK repo's `README.md` and `examples/`.

## Questions & community

- **GitHub Discussions:** https://github.com/orgs/thevoltcloud/discussions — usage questions, ideas, show-and-tell.
- **Question issue template:** open a `Question` issue on the relevant repo if it's specific to that codebase.

Please search existing discussions and issues before opening a new one.

## Bugs & feature requests

Open an issue on the **specific repository** using the bug or feature template. For general or cross-repo topics, use Discussions.

## Security

Do **not** use public issues for security reports. See [`SECURITY.md`](SECURITY.md) — email `security@voltcloud.ai`.

## Paid / commercial support

Volt customers (Spark, Forge, Vault) get support through their contract SLA. Reach your account contact, or email **support@voltcloud.ai**. For sales, **hello@voltcloud.ai**.

## What's not supported here

Closed-weights model hosting (Claude, GPT, Gemini) is not part of Volt — those questions are out of scope for this org.
